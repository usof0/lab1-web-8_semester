# JavaScript Pipeline CLI

Консольное приложение на JavaScript (Node.js), которое получает данные из публичного API, обрабатывает JSON и пропускает данные через простой конкурентный pipeline.

---

## Описание проекта

Программа выполняет следующие действия:

1. Делает HTTP-запрос к публичному API  
   `https://jsonplaceholder.typicode.com/posts`

2. Получает и парсит JSON-ответ

3. Пропускает данные через pipeline из двух этапов:
   - **Этап 1:** получение и подготовка данных
   - **Этап 2:** фильтрация и преобразование данных

4. Использует конкурентную обработку:
   - **fan-out** — распределение данных между несколькими workers  
   - **fan-in** — объединение результатов в один поток

5. Принимает аргументы из командной строки и выводит результат в stdout

---

## Используемые технологии

- JavaScript (Node.js ≥ 18)
- Встроенный `fetch` для HTTP-запросов
- Асинхронность (`async/await`, `Promise.all`)
